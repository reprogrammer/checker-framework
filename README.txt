This is the checker-framework repository.

This repository contains several related projects:

 checkers    the Checker Framework proper
 demos       slides and source code for demos of the Checker Framework
 javaparser  a parser for Java source code; supports type annotations
 release     buildfiles for making a release


Documentation for installing and using the Checker Framework appears in the
Checker Framework manual, which you can find in directory checkers/manual/
and online at
http://types.cs.washington.edu/checker-framework/current/checkers-manual.html
